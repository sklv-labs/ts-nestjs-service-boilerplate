# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-XX

### Added

- Initial release of @sklv-labs/ts-nestjs-service-boilerplate
- Basic NestJS application structure with `AppModule`, `AppController`, and `AppService`
- Comprehensive TypeScript configuration with path aliases
- Full test setup with Jest including unit and E2E test configurations
- ESLint and Prettier configuration
- Development scripts for hot reload and debugging
- Production build configuration

### Features

- **Type-Safe**: Full TypeScript support with path aliases (`@/*`)
- **Test Ready**: Pre-configured Jest setup for unit and E2E tests
- **Development Tools**: Hot reload, debugging, and linting support
- **Organized Structure**: Server code in `server/src`, docs at root level
- **NestJS Native**: Built on top of NestJS with best practices
